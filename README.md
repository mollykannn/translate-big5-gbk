# 繁簡轉換

繁簡轉換線上版 - [https://mollykannn.github.io/translate-big5-gbk](https://mollykannn.github.io/translate-big5-gbk)

繁簡辭庫出自:
- [tongwen-dict](https://github.com/tongwentang/tongwen-dict/)
(11/06/2022: 更新了使用github上的繁簡轉換辭庫)

## 安裝 (Install)

```shell
yarn install
```

## 使用方法 (Usage)

建立檔案 (Create files)
```shell
yarn build
```

運行 (Run)
```shell
yarn dev
```
